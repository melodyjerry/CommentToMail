# CommentToMail

1. 基于原V3.1.0版本重新编写

2. 更新了PHP Mailer版本

3. 优化了使用SMTP发信的证书认证

4. 修复使用QQ邮箱（非企业邮箱）的时候会发现邮件发不出去的BUG

5. 将异步触发更换为网址监控运行

6. 修复数据库导入时偶发性的“Database Query Error” 

7. SMTP 加入 TLS 支持（目前支持 SSL、TLS)

8. 新增反垃圾邮件策略

**支持的 Typecho 版本 >=1.0**
